Our World Time is a real-time world clock and time zone converter. 

I built this utilizing the interactive dial to find the current local time in cities from around the globe, perfect for scheduling international meetings or keeping in touch with friends and family abroad. Swipe up to favorite timezone, swipe down for local timezone.

### Key Features
* Dynamic theming that updates based on the local time of the selected city.
* Built utilizing the modern `Intl` API for accurate, DST-aware time conversions.
